Fonctionnement

LAWRIDER génère - s'il n'existe pas - un flux RSS pour un texte qui vous intéresse. Le flux est constitué des textes parus au Journal officiel faisant mention de ce texte. Les données du flux sont issues de la base Légifrance. Ils sont hébergés sur le site et mis à jour toutes les heures.

Le code source est disponible sur github.com/pantonin/. Le format des flux est le RSS 2.0. Le format atom sera intégré ultérieurement.


Flux RSS

Un flux RSS est un format de données utilisé pour la syndication de contenus. Les flux RSS existent depuis 1999 et s'ils n'en finissent pas de mourir c'est qu'ils sont particulièrement utiles : système pull plutôt que push, simple, utile et efficace.

Les flux RSS peuvent être lus sur un client mail (Outlook ou Thunderbird) mais il est aussi possible - et recommandé - d'utiliser un logiciel spécifique. À cet égard je recommande QuiteRSS qui est simple d'utilisation, open-source et disponible sur toutes les plateformes.


LAWRIDER est développé par Antonin Paillet.
